getSpell

  • getSpell(url, modifier?): Observable<SpellI>

  • This function creates a SpellI by making an API request given a specific url. DnD API returns a specific url with each spell name when searching all spells that contains additional information about the spell. This function makes a separate API call using that new url, and attempts to create a SpellI data type using the info from the API request

    Parameters

    • url: string

      url from SpellResponseResults type

    • modifier: number = 0

      In order to properly set spell damage, the spell modifier needs to be set based on what the user had inputted

    Returns Observable<SpellI>

    Observable that when subscribed to will give a Spell data type that holds all necessary information about the spell
